# Te-laat-komers CRUD

Beschrijving
Dit project registreert studenten die te laat zijn.
De applicatie maakt, leest, wijzigt en verwijdert registraties.
De interface geeft een duidelijk overzicht met alle ingevoerde gegevens.
Het systeem gebruikt het Yii2 framework.

Functies
Student toevoegen
Studentgegevens wijzigen
Student verwijderen
Overzicht tonen
Sorteren op kolommen
Validatie van invoer

Installatie
Download of clone het project.
Installeer afhankelijkheden via Composer.
Maak een MySQL database.
Importeer het schema of gebruik de migrations van Yii2.
Pas de databaseconfiguratie aan in config/db.php.
Start de ontwikkelserver met php yii serve.

Structuur
Controllers bevatten de CRUD-logica.
Views tonen de tabellen en formulieren.
Models valideren invoer en communiceren met de database.
De projectstructuur volgt de Yii2 MVC standaard.
